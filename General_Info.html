General Info


Ram is essentially a short term data storage, it is used for storing information about the current application or document that you are using, when you turn off your pc the ram all the data on the ram is deleted. Ram stands for Random Access Memory and it used instead of regular storage devices like harddrive and ssds because accessing the information is much faster
