DDR

What is DDR RAM?

DDR SDRAM or Double Data Rate
Synchronous Dynamic Random-Access Memory
is the standard memory that we use.

There have beem many iterations of the DDR line of RAM
 The most recent is DDR4 with DDR5 comming
to the commercial market soon

Normal DDR RAM is used in most standard laptops and computers

DDR4 is the current most recent iteration of the DDR line of RAM.
 There are two formats for DDR RAM, SODIMM (Small Outline Dual In-line Memory Module)
and DIMM (Dual In-line Memory Module)

DDR RAM is labled by generation then clock speed along with size,
e.g., DDR4-3200 8gb where DDR4 is the generation, 3200 is the clockspeed and 8gb is the size
Typicaly, the cost of each subsequent generation decreases over time
but depending on the clockspeeds and size the price will be respectively lower or higher

What is it Used for?

DDR RAM is used in standard computers and laptops
With the DIMM form factor being used in computers
and the SODIMM form factor in laptops.

Laptops uses SODIMMs to load applications

Tower PC's use memory DIMMs to load applications

The History of DDR Memory

Starting in 1993 when the first iteration of DRAM was introduced

The next iteration was in 2000 with DDR SDRAM

After that in 2003 with DDR2 SDRAM

Later on in 2007 came DDR3 SDRAM

Finaly the current iteration came out in
2014 which is DDR4 SDRAM
